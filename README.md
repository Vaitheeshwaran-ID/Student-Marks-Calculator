# Student Mark Calculator

A Java console application that calculates student averages, 
assigns grades, and identifies the topper from a list of students.

## Features
- Calculate average marks for each student
- Assign grades (A, B, C, D, F) based on average
- Display Pass/Fail result for each student
- Identify and display the topper by name

## How to Run
1. Compile: `javac Main.java Student.java`
2. Run: `java Main`
3. Enter number of students
4. Enter name and 3 subject marks for each student

## Sample Input

3
vaithee 90 85 95
siva 70 65 60
kumar 40 45 35
